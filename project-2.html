Project Overview

The Student Info app is designed for teachers to easily manage and track student details with a user-friendly interface. It features three main screens: the Add Student screen allows teachers to input comprehensive student information, including name, age, school standard, section, parent contact number, address, and blood group. The View Student List screen provides quick access to all stored student details in an organized manner. Additionally, the Map screen displays student locations, making it easy for teachers to see where their students live.

With secure authentication for login and signup, the app ensures that your data is protected while offering a smooth and efficient way to handle student information.
